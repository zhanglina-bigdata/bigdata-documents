# 清洗列表



| 垃圾字符      | 中文描述       |
| ------------- | -------------- |
| ？或?         | 去掉           |
| ！或!         | 去掉           |
| *             | 去掉           |
| $             | 去掉           |
| #             | 去掉           |
| .             | 去掉首尾.      |
| -             | 去掉尾         |
| =             | 去掉           |
| +             | 去掉           |
| （）          | 中文转成英文() |
| 十进制unicode | 转成汉字       |
| 空格          | 去掉首尾空格   |

# 清洗方法

### 十进制unicode  转成汉字

- 导入common-lang-2.x.jar包，利用其中的StringEscapeUtils.unescapeHtml(str);方法
